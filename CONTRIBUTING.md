# Berkontribusi pada OuroLang

Terima kasih sudah tertarik berkontribusi pada OuroLang! 🎉

## Cara Memulai

### 1. Fork & Clone
```bash
git clone https://github.com/YOUR_USERNAME/OuroLang.git
cd OuroLang
```

### 2. Setup Development
```bash
make          # Kompilasi VM
./ouro examples/halo.ouro  # Test
```

### 3. Buat Branch untuk Fitur/Bug Fix
```bash
git checkout -b feature/nama-fitur
# atau
git checkout -b fix/nama-bug
```

## Area Kontribusi yang Kami Butuhkan

## Submission Guidelines

### Issue
1. Check existing issues dulu
2. Describe problem dengan jelas
3. Include reproduction steps kalau bug

### Pull Request
1. Reference issue yang relevant
2. Describe changes dengan detail
3. Include test cases
4. Keep commits atomic dan descriptive

## Coding Standards

- **Language:** C (untuk VM), Ouro (untuk compiler components)
- **Comments:** In Bahasa Indonesia (optional), code should be self-documenting
- **Format:** Follow existing code style
- **Testing:** Add tests untuk new features

## Communication

- **English OK** - Tapi Bahasa Indonesia juga welcome
- **Be respectful** - Ini hobby project, semua orang learning
- **Ask questions** - No such thing as stupid questions
- **Share ideas** - Feature requests dan suggestions welcome
